# Forum Application

## Overview
This is a web-based forum application built using Flask and MongoDB. Users can register, log in, create posts, reply to posts, and like content. The application provides a platform for discussions and content sharing.

## Features
- User registration and login
- Create and view posts
- Reply to posts
- Like/unlike posts and replies
- Responsive design for mobile and desktop

## Installation
1. Clone the repository:
	git clone https://github.com/HamreethLS/Forum-Using-Flask.git
2. Navigate to the project directory:
	cd forum
3. Install the required packages  
	pip install -r requirements.txt
4. Set up your MongoDB database and update the connection string in `app.py` if necessary.

## Running the Application
To run the application, execute:

	pyhton app.py
or 
	flask run

Visit `http://127.0.0.1:5000` in your web browser to access the forum.
